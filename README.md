# ProxyControl

**ProxyControl** — это Windows-приложение для удобного управления прокси-серверами и маршрутизацией сетевого трафика. Программа позволяет легко настраивать, переключать и мониторить прокси-соединения прямо из графического интерфейса.

## 🎯 Для чего нужен этот проект?
Проект предназначен для пользователей, которым необходимо гибко управлять своим сетевым подключением: обходить сетевые ограничения, разделять трафик для конкретных приложений или просто быстро переключаться между различными SOCKS5/TCP прокси без необходимости каждый раз лезть в системные настройки Windows.

## ⚙️ Как это работает и что умеет?
ProxyControl работает как локальный прокси-клиент и менеджер правил. Он перехватывает и перенаправляет сетевой трафик в зависимости от пользовательских настроек.

**Основные возможности:**
* **Поддержка различных протоколов:** Работа с SOCKS5, TCP и DNS проксированием.
* **Режим TUN-интерфейса:** Возможность завернуть весь трафик системы через виртуальный сетевой интерфейс (`TunService.cs`).
* **Управление системным прокси:** Автоматическое изменение системных настроек Windows (`SystemProxyHelper.cs`).
* **Мониторинг:** Отслеживание сетевого трафика в реальном времени и мониторинг активных процессов (`TrafficMonitorService.cs`, `ProcessMonitorService.cs`).
* **Правила маршрутизации:** Настройка правил для определенных адресов или приложений.
* **Автообновления:** Встроенная система проверки и загрузки обновлений напрямую с GitHub (`GithubUpdateService.cs`).
* **Удобный интерфейс:** Современный дизайн с поддержкой темной темы (`DarkTheme.xaml`).

## 🛠 Технологии и инструменты
Проект разработан под ОС Windows с использованием современного стека технологий от Microsoft:
* **Язык программирования:** C#
* **Платформа:** .NET
* **Графический интерфейс:** WPF (Windows Presentation Foundation) с использованием XAML
* **Архитектура/Паттерны:** MVVM (Model-View-ViewModel)

## 🚀 Запуск и сборка
1. Склонируйте репозиторий.
2. Откройте решение `ProxyControl.sln` в Visual Studio (или Rider).
3. Соберите проект (Build).
4. Для корректной работы некоторых сетевых функций (например, режима TUN) приложению могут потребоваться права Администратора. Вы также можете использовать скрипт `1rebuild.bat` для быстрой пересборки.
